# URSSAF-new-tab-file
Extension chrome qui permet d'ouvrir les pièces jointes des messages reçus sur le site de l'[URSSAF](https://www.autoentrepreneur.urssaf.fr "URSSAF") dans un nouvel onglet au lieu de devoir les télécharger.
